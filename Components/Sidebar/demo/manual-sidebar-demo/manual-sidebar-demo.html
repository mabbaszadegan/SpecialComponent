دموی منوی دستی

در این دمو، منوی sidebar به صورت دستی با استفاده از تگ‌های HTML تعریف شده است

کنترل‌های دمو:

ویژگی‌های این دمو:

  • تعریف دستی منو: تمام آیتم‌های منو با استفاده از تگ‌های HTML تعریف شده‌اند
  • ساختار سلسله‌مراتبی: پشتیبانی از منوهای چند سطحی
  • ویژگی‌های پیشرفته: آیکون، توضیحات، نشان‌ها (badge) و رفتارهای مختلف
  • کنترل‌های تعاملی: امکان افزودن، حذف و مدیریت آیتم‌ها
  • حفظ وضعیت: به خاطر سپردن وضعیت باز/بسته بودن آیتم‌ها
  • جستجو: قابلیت جستجو در آیتم‌های منو

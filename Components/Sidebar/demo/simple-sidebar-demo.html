کامپوننت سایدبار ساده

این یک نمونه ساده از کامپوننت سایدبار است که نشان می‌دهد چگونه می‌توان یک منوی ناوبری زیبا و کاربردی ایجاد کرد.

🎯 ویژگی‌های این کامپوننت

  • طراحی زیبا و مدرن با انیمیشن‌های نرم
  • پشتیبانی از آیکون‌های FontAwesome
  • نمایش توضیحات و badge برای هر آیتم
  • پشتیبانی از آیتم‌های تو در تو (nested)
  • انیمیشن hover و کلیک
  • طراحی responsive و mobile-friendly

💡 نحوه استفاده

برای استفاده از این کامپوننت در پروژه‌های خود:

  1. فایل CSS را در پروژه خود قرار دهید
  2. ساختار HTML مشابه را کپی کنید
  3. JavaScript مورد نیاز را اضافه کنید
  4. آیتم‌های منو را مطابق نیاز خود تغییر دهید

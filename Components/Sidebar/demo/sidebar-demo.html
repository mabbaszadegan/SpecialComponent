Special Sidebar Component - Enhanced with Data Source Support

کامپوننت سایدبار با پشتیبانی از منبع داده و HTML دستی

🎯 ویژگی‌های جدید

این کامپوننت بهبود یافته قابلیت‌های زیر را ارائه می‌دهد:

📊 پشتیبانی از منبع داده

بارگذاری منوها از فایل JSON یا API endpoint

🖊️ تعریف دستی HTML

امکان تعریف آیتم‌ها به صورت HTML درون کامپوننت

🔄 بروزرسانی خودکار

بروزرسانی خودکار داده‌ها در فواصل زمانی مشخص

⏳ نمایش وضعیت بارگذاری

نمایش spinner و پیام بارگذاری هنگام دریافت داده

🔧 نحوه استفاده

// روش 1: استفاده از منبع داده <sc-sidebar data-source="/api/menu.json"></sc-sidebar> // روش 2: استفاده از API <sc-sidebar api-endpoint="/api/menu" refresh-interval="30000"></sc-sidebar> // روش 3: تعریف دستی HTML <sc-sidebar> <sc-sidebar-item key="dashboard" text="داشبورد" icon="fa-solid fa-tachometer-alt"> <sc-sidebar-item key="overview" text="نمای کلی"></sc-sidebar-item> </sc-sidebar-item> </sc-sidebar>

📊 استفاده از منبع داده

کامپوننت می‌تواند منوها را از منابع مختلف بارگذاری کند:

📁 فایل JSON

بارگذاری از فایل JSON محلی یا URL

🌐 API Endpoint

دریافت داده از API با قابلیت بروزرسانی خودکار

🔄 بروزرسانی خودکار

تنظیم فاصله زمانی برای بروزرسانی داده‌ها

📝 ساختار داده JSON

{ "items": [ { "key": "dashboard", "text": "داشبورد", "icon": "fa-solid fa-tachometer-alt", "description": "نمای کلی سیستم", "behavior": "always-open", "children": [ { "key": "overview", "text": "نمای کلی", "icon": "fa-solid fa-chart-line", "behavior": "clickable" } ] } ] }

🎮 کنترل‌های منبع داده

🖊️ تعریف دستی HTML

امکان تعریف آیتم‌ها به صورت HTML درون کامپوننت:

🏷️ ویژگی‌های آیتم

key, text, icon, description, badge, disabled, behavior, category

🌳 ساختار تو در تو

پشتیبانی از آیتم‌های تو در تو با استفاده از sc-sidebar-item

📝 مثال HTML

<sc-sidebar id="manualSidebar"> <sc-sidebar-item key="dashboard" text="داشبورد" icon="fa-solid fa-tachometer-alt" description="نمای کلی سیستم" behavior="always-open"> <sc-sidebar-item key="overview" text="نمای کلی" icon="fa-solid fa-chart-line" behavior="clickable"> </sc-sidebar-item> <sc-sidebar-item key="analytics" text="تحلیل‌ها" icon="fa-solid fa-chart-bar" badge="جدید" behavior="clickable"> </sc-sidebar-item> </sc-sidebar-item> <sc-sidebar-item key="settings" text="تنظیمات" icon="fa-solid fa-cog" behavior="clickable"> </sc-sidebar-item> </sc-sidebar>

🎮 کنترل‌های HTML دستی

🌐 استفاده از API

کامپوننت می‌تواند داده‌ها را از API دریافت کند:

🔗 API Endpoint

تنظیم URL برای دریافت داده از API

📋 Headers

تنظیم headers برای احراز هویت و سایر نیازها

⏰ بروزرسانی خودکار

تنظیم فاصله زمانی برای بروزرسانی خودکار

📝 مثال API

// تنظیم API endpoint sidebar.scSidebarSetAPIEndpoint('/api/menu', { 'Authorization': 'Bearer token123', 'Content-Type': 'application/json' }); // تنظیم بروزرسانی خودکار (هر 30 ثانیه) sidebar.scSidebarSetRefreshInterval(30000); // بروزرسانی دستی sidebar.scSidebarRefreshData();

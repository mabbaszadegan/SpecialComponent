ویژگی‌های این دمو:

  • بارگذاری از JSON: منو از فایل JSON خارجی بارگذاری می‌شود
  • ساختار پویا: امکان تغییر ساختار منو بدون تغییر کد

  • بارگذاری خودکار: باز شدن خودکار آیتم‌های مهم
  • حفظ وضعیت: به خاطر سپردن وضعیت آیتم‌ها

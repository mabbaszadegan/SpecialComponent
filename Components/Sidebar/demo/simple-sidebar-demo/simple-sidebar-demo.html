دموی Sidebar با قابلیت باز و بسته شدن

نمایش قابلیت‌های جدید expand/collapse برای منوهای چند سطحی

کنترل‌های دمو:

ویژگی‌های جدید expand/collapse:

  • منوهای چند سطحی: پشتیبانی از منوهای عمیق با قابلیت باز و بسته شدن
  • تشخیص خودکار: تشخیص خودکار آیتم‌های قابل باز شدن بر اساس زیرمجموعه‌ها
  • انیمیشن‌های نرم: انیمیشن‌های CSS برای باز و بسته شدن منوها
  • حفظ وضعیت: حفظ وضعیت باز/بسته منوها در localStorage
  • فلش‌های راهنما: نمایش فلش برای آیتم‌های قابل باز شدن
  • رفتار هوشمند: تشخیص خودکار رفتار آیتم‌ها (expandable یا clickable)

نمایش ویژگی‌ها:

منوهای چند سطحی

پشتیبانی از منوهای عمیق با انیمیشن‌های نرم

قابلیت باز و بسته شدن

باز و بسته شدن منوها با کلیک روی آیتم‌های قابل گسترش

فلش‌های راهنما

نمایش فلش برای آیتم‌های قابل باز شدن

حفظ وضعیت

حفظ وضعیت باز/بسته منوها در localStorage

نحوه استفاده:

1

کلیک روی آیتم‌های قابل باز شدن

روی آیتم‌هایی که فلش دارند کلیک کنید تا زیرمجموعه‌ها نمایش داده شوند

2

استفاده از دکمه‌های کنترل

از دکمه‌های "باز کردن همه" و "بستن همه" برای مدیریت منوها استفاده کنید

3

جستجو در منوها

از قابلیت جستجو برای پیدا کردن آیتم‌های خاص استفاده کنید

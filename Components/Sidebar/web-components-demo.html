üöÄ Features & Capabilities

  • Themes: Light and dark theme support with CSS variables
  • Positioning: Left or right sidebar positioning
  • Customizable Width: Set any width value (px, %, em, etc.)
  • Collapsible: Expandable/collapsible sidebar with toggle button
  • Header/Footer: Optional header and footer sections
  • Searchable: Built-in search functionality for items
  • Animations: Smooth transitions and animations
  • Responsive: Mobile-friendly with overlay support
  • State Persistence: Remember collapsed state and search queries
  • Nested Items: Support for hierarchical menu structures
  • Font Awesome Icons: Integrated icon support
  • Keyboard Navigation: Keyboard accessibility support
  • Custom Events: Event dispatching for item clicks
  • Default Items: Automatic creation of sample items
  • Shadow DOM: Encapsulated styling and structure
  • Web Components API: Modern browser standards compliance

üîß How It Works

  1. Web Components: Built using Custom Elements API and Shadow DOM
  2. Attribute-based Configuration: All features controlled via HTML attributes
  3. State Management: Internal state management with localStorage persistence
  4. Event Handling: Comprehensive event system for user interactions
  5. Responsive Design: CSS Grid and Flexbox for modern layouts
  6. Theme System: CSS custom properties for easy theming
  7. Accessibility: Keyboard navigation and ARIA support
  8. Performance: Efficient rendering and minimal DOM manipulation

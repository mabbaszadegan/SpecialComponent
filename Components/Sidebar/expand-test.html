Sidebar Expand/Collapse Demo

This demo shows the expand/collapse functionality of sidebar menu items. Click on items with children to expand/collapse them.

Features:

  • ✅ Automatic detection of expandable items
  • ✅ Smooth animations with CSS transitions
  • ✅ State persistence (remembers expanded items)
  • ✅ Keyboard navigation support
  • ✅ Responsive design
  • ✅ Theme support (light/dark)

How to Use

The sidebar automatically detects which items can be expanded based on their children.
<sc-sidebar theme="light" show-header="true" collapsible="true" searchable="true" remember-state="true"> <sc-sidebar-item key="users" text="User Management" behavior="expandable"> <sc-sidebar-item key="user-list" text="User List" behavior="clickable"> </sc-sidebar-item> <sc-sidebar-item key="add-user" text="Add User" behavior="clickable"> </sc-sidebar-item> </sc-sidebar-item> </sc-sidebar>

Behavior Types:

  • expandable - Item can expand to show children
  • clickable - Item can be clicked for action
  • auto - Automatically determined based on content

Key Features:

  • Automatic arrow indicators for expandable items
  • Smooth CSS transitions for expand/collapse
  • State persistence across page reloads
  • Responsive design for mobile devices
  • Search functionality to filter items
  • Keyboard navigation support
